# Telegram TXT Sender

Простой Python-скрипт для отправки текста из `.txt` файла в выбранный Telegram-чат через бота.

## Требования

- Python 3.x
- Библиотека `requests`

Установить `requests`:

```bash
pip install -r requirements.txt
```
## Использование
Создайте Telegram-бота через BotFather и получите токен.

Узнайте chat_id нужного чата.

Подготовьте текстовый файл .txt с содержимым для отправки.

Запуск скрипта:

```bash
python main.py
```
При запуске скрипт попросит:

Путь к .txt файлу

chat_id

После ввода скрипт отправит содержимое файла в указанный чат.

Примечания
Бот должен иметь доступ к чату.

Скрипт не поддерживает большие файлы (>4096 символов на одно сообщение).
